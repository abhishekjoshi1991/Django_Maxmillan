Ran Application Template

    Django looks for templates in setting.py file in templates location
    there are two main parameter
  • DIRS
  • APP_DIRS
    To use APP_DIRS app must be in list of installed apps
    APP_DIRS if set to true, automatically fetches all templates from applications folder in application it searches for templates folder from there it loads templates
    Convention is to create templates folder then folder with app name inside it all files are present
    DIRS on other hand loads templates those are not in applications, for that we need to give path inside DIRS like BASE_DIR + '/templates'
    for app specific templates we do not need to enter full path and write it into DIRS we already have APP_DIRS for that
